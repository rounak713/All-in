## 🚀 Overview

**All in** is a futuristic, premium SaaS platform that redefines networking and professional identity. Designed to look and feel like a billion-dollar startup, the application features extreme minimalism, deep dark modes, and buttery-smooth animations. It is built for students, creators, professionals, and enterprise teams to seamlessly share their portfolios, analytics, and contact information.

## ✨ Features

- **Digital Profiles:** Auto-generated, highly customizable profiles with Apple-quality dark mode themes.
- **AI Intelligence:** Meet your AI Identity Assistant—instantly craft professional bios, portfolios, and cover letters.
- **Rich Analytics:** Real-time dashboards to track card taps, link clicks, and audience geography.
- **Enterprise Ready:** Bulk provisioning, SSO security, global template enforcement, and CRM integrations (Salesforce, Hubspot, etc.).
- **Smart Networking:** Share your info via an NFC metal card or a custom QR code. Allow clients to book meetings and make payments directly from your profile.

## 🛠 Tech Stack

- **Framework:** [Next.js 16 (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** TypeScript

## 🏎 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/dooti2325/All-in.git
   cd "All-in/website"
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Experience the Magic:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🎨 Design Philosophy
*   **Color Palette:** Absolute Black (`#000000`), Off-White (`#F5F5F5`), All-in Red (`#FF1A1A`), and Slate Gray (`#7A7A7A`).
*   **Typography:** A beautiful pairing of modern sans-serif (Inter) and elegant display serif (Playfair Display).
*   **Aesthetics:** Glassmorphism, subtle red glows, micro-interactions, and premium layouts.
